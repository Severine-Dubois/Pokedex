# Pokedex

## Composer/Feature used :
- Altorouter
- Var-dumper
- Autoload
- Alto-dispatcher

## 1st Gen
- Displays all 151 first Pokémons, using a local database (no API)
- Show one of them with its stats + description
- Can list them by type
- Includes a dark mode
